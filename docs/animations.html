Animations

A handful of animations including bounce ins, fade ins, zoom ins and flipping. Make your website / web app look even better than ever with this set of awesome animations.

Overview

Make your website pop with these cool animations thanks to Animate.css by Daniel Eden.

All Animations are stored within less/base/animations/.

We've included a core set of animations to get you started:

  • _bounce.less
  • _fade.less
  • _flip.less
  • _zoom.less

Below is a list of the current animations.
