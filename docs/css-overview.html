Base CSS

All the stylesheets are contained within the less/ directory and WinLess should be used to export the final output to the root / directory

The main CSS file that should be edited is style.less and the structure is is set out neatly with guided comments

Body and Typography

Base has a global default font size of 14px (which is converted into rem units for scalability and accessibility). This is applied to the <body> and all <p> tags.

NOTE: rem units are not supported in some browsers, so a px fallback occurs.
