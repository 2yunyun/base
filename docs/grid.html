The Grid System

The powerful grid system uses a variation of the 960 grid system. The syntax is super simple and it works accross all major browsers including Chrome, Safari, Firefox and even Internet Explorer 7+.

For a better understanding of how the grid system works, please reivew the examples and comments below.

Example 1

This is the most basic example of how to use the grid.
7
5

Code

1  ## Example 1 ##
3  <!-- wrap columns within a .section or .clear div to clear the floating columns -->
4  <div class="section row">
5
6    <!-- Add a .col class followed with the column width in word value (one, two, three..., twelve) -->
7      <div class="col seven">7</div>
8      <div class="col five">5</div>
9
10  </div> <!--/section or clear -->

Example 2

To make columns full width for Tablet, simply add the .tablet-full class
7
5

Code

1  ## Example 2 ##
3  <!-- wrap columns within a .section or .clear div to clear the floating columns -->
4  <div class="section row">
5
6    <!-- Add the .tablet-full class to make columns 100% width on tablet view -->
7      <div class="col seven tablet-full">7</div>
8      <div class="col five tablet-full">5</div>
9
10  </div> <!--/section or clear -->

Example 3

Furthermore, you can make columns full width on Mobile by adding the .mobile-full class.

10  </div> <!--/section or clear -->
