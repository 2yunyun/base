The Grid

The powerful grid system uses a variation of the 960 grid system. The syntax is super simple and it works accross all major browsers including Chrome, Safari, Firefox and even Internet Explorer 7+.

A Basic Grid Example

Start by adding a parent element with a class of .clear as this will ensure all the columns are cleared correctly.

You can then add elements with a .col class to define a column.

Column widths can then be specified with the .mobile-col-#, .tablet-col-# and .col-# classes.

* Column width helpers can be used without .col to resize elements accordingly.

Desktop Column 4 Tablet Column 9 Mobile Column 1-2
Desktop Column 8 Tablet Column 3 Mobile Column 1-2 
<!-- Grid Example -->
<div class="clear">
  <div class="col col-4 tablet-col-9 mobile-col-1-2">
    Column Content
  </div>
  <div class="col col-8 tablet-col-3 mobile-col-1-2">
    Column Content
  </div>
</div>

Grid Helpers

In addition to the above example, a full list of all column helpers are listed below.

Grid Classes
Class / Helper Result
.mobile-full sets column width to 100% and removes floats for mobile devices
.tablet-full sets column width to 100% and removes floats for tablet devices
.col-1-2 sets column width to 50% for all devices
.col-1-3 sets column width to 33% for all devices
.col-1-4 sets column width to 25% for all devices
.tablet-col-1-2 sets column width to 50% for tablet devices
.tablet-col-1-3 sets column width to 33% for tablet devices
.tablet-col-1-4 sets column width to 25% for tablet devices
.mobile-col-1-2 sets column width to 50% for mobile devices
.mobile-col-1-3 sets column width to 33% for mobile devices
.mobile-col-1-4 sets column width to 25% for mobile devices
