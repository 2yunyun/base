HTML Template

A 99% fat free HTML template which includes the core essentials to get you started on your next web project.

Break down of the template file:

Language Attribute

By default, the language attribute lang is set to English en. It is recommended you update this to your local language.

NO-JS Class

By default, the html template has a .no-js class applied to the html tag. When JavaScript is active, Modernizr will re-write this class to .js. This allows fallback styles to be added quite easily when JavaScript is disabled.

X-UA-Compatible

Makes sure the latest rendering mode for IE is triggered.

Mobile Viewport

Ensures scale settings remain consistent when switching orientations on phones.

Favicons

The standard favicon for desktop devices and 3 icons for mobile and tablet devices.

Custom Modernizr

A custom build of Modernizr is included which checks support for JS, SVG, touch support and adds appropriate classes to the html element if the features are supported or unsupported.

jQuery (Google CDN)

The latest jQuery version is included (referenced from Google's CDN) along with a local fallback for offline development.

Google Analytics

An optimized version of Google Analytics taken from HTML5 Boilerplate.

That's all folks!
