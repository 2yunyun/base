Assets

Store your reusable assets and components in this special file.

Overview

By default, the assets file is very small and has a few components to get you started.

You can populate the assets file with your own reusable modules and components.

Base includes a few assets which are as follows:

  • Custom Buttons
  • Custom Select

Custom Buttons

Base includes a basic button to get you started.

You can add more within the _assets.less or _assets.scss file or modify the existing button helpers (.button-primary, .button-secondary).

Default Button

Primary Button
Secondary Button

Code Snippet

<a href="#" class="button">Default Button</a>
<a href="#" class="button button-primary">Primary Button</a>
<a href="#" class="button button-secondary">Secondary Button</a>
<a href="#" class="button button-link">Button styled as a link</a>
